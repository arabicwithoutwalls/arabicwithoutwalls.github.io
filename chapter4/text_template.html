Keep in mind...

  • Each question is encapsulated by a <p> tag. In this example, it is just a plain <p> tag, but if everything is in Arabic you might want to adjust the alignment and text direction (i.e. align="right" dir="rtl").
  • The answers and hints are marked up by hidden <span> tags. Arabic strings within those tags are also place within <span> tags of class="arabic_lg". If the answer is an English string and not Arabic, the <span class="arabic_lg"> is not necessary. Place it inside of a normal span tag with no attributes. If the answer is not inside of a span tag, it will cause an error in the script. Look at the code for question 2 versus the code fro question 1.
  • Adding or removing spaces may throw off the script. The <p> tag is the parent node, and each tag (or other element) inside the <p> tag represents a child node. Adding or removing spaces will throw off the order of those nodes. This also means that the hidden hint (the second hidden span) needs to remain in every question, whether there is a hint for it or not. The link that says Hint can be removed, but the hidden hint <span> tag must remain.
  • If the text field (input type="text") is intended for Arabic strings, you will want to add the "dir=rtl" attribute to the tag. The length of the text field can be adjusted with the size attribute (i.e. size="30"), and the size of the text can be adjusted with the style attribute (i.e. style="font-size: 16px;"). Note: Make sure that all of the text fields on the page have the same font size. 16px seems like a good medium--the English isn't too big, nor the Arabic too small.
  • The second argument passed into the show_hidden_answer( ) function is the question number. This needs to be an English integer, even if numeration on the side is Arabic.
