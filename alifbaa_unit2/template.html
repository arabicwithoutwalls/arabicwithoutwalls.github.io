Revealing Hidden Text

This text was hidden.The simplest way to reveal hidden text is with a link such as this.

This hidden text trick is pretty simple. The first thing in the P tag (it also works with LI and should with span) is the hidden text. It's in a span tag of class="hidden," which is declared as not displayed in the stylesheet. Somewhere in the LI tag is a link. The JavaScript in this link copies the innerHTML of the hidden span and appends it to the innerHTML of the link's parent element (LI in this case).

One warning: There can be no whitespace (spaces or new lines) between the first P tag and the hidden span tag.

If you think that it would be more intuitive to put the hidden span at the end and simply change its style.display attribute, you're right. However, because IE counts nodes incorrectly, the hidden span's place in the P's childNodes array will be different than in Firefox.
